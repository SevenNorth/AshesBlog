使用`clean-webpack-plugin`自动清理构建目录，避免每次构建前都需要手动删除dist。
默认会删除output指定的输出目录

<!-- more -->

```js
const { CleanWebpackPlugin } = require('clean-webpack-plugin');

module.exports = {
  // ....
  plugins: [
    new   CleanWebpackPlugin()
  ]
};
```
